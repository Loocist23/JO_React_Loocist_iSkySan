# JO_React_Loocist_iSkySan 
JO_React_Loocist_iSkySan is a course project in react for YNOV MONTPELLIER
## Prerequisites

- Node.js
- npm
- React
- React-Router
- Axios
- React-Icons

## Installation

```bash
npm install
```

## Usage

```bash
npm run start
```
